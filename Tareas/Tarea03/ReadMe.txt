Para compilar el Programa1 nos dirigimos dentro de la terminal al directorio donde se encuentra el archivo .c
/Tarea3/src/ y escribimos lo siguiente en la terminal para compilar.
> gcc Programa1.c -o Programa1 -lpthread -lm

Posterior a eso se generara un archivo llamado 'Programa2' el cual es el ejecutable.
Para ejecutar el programa que compilamos anteriormente, escribimos lo siguiente en la terminal.
> ./Programa1
