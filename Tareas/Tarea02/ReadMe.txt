Para compilar los programas nos dirigimos dentro de la terminal al directorio donde se encuentrar los archivos .c
/Tarea2/src/ y escribimos lo siguiente en la terminal para compilar.
> gcc ProgramaN.c donde N es el numero del programa.

Posterior a eso se generara un archivo llamado 'a.out' el cual es el ejecutable.
Para ejecutar el programa que compilamos anteriormente, escribimos lo siguiente en la terminal.
> ./a.out
